==================================
ΣΗΜΑΝΤΙΚΑ ΑΡΧΕΙΑ
==================================

* pages/Shop.js: Σελιδα που ανοιγει στο url "/shops/:id". Χρησιμοποιει ενα ShopInfo για να δειξει τις πληροφοριες
* components/ShopInfo.js: Δειχνει τις σελιδες του καταστηματος. Εχει και ενα πινακα με τα προιοντα του, ανοιγει ενα ShopAddProductPriceModal για καταχωριση νεου
* components/ShopAddProductPriceModal.js: Popup για επιλογη προϊοντος/τιμης/ημερομηνιων για καταχωρηση

* pages/ShopListing.js: Λιστα με ολα τα καταστηματα σε πινακα. Οι συνδεδεμενοι χρηστες μπορουν και να τα επεξεργαζονται. Πατωντας πανω στο ονομα του καθενος παμε στη σελιδα του. Μπορει να ανοιγει ενα ShopModal για εισαγωγη νεου καταστηματος
* components/ShopModal.js: Χρησιμοποιειται ειτε για δημιουργια νεου προιοντος ειτε επεξεργασια υπαρχοντος.

* utils/tags.js: Συναρτησεις για μετατροπη των λιστας απο tags απο και προς string
